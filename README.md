## Technologies used
- Virus Scanning: VirusTotal
- Authentication: JWT
- Frontend: Next js
- Storage: Firebase
- Styles: Tailwind CSS
- Validation: Yup
- Forms: Formik
- Animation: Framer
- Database: Mongo DB(NoSQL)
- Email: Email js
